② 類題

左に例題を入れて「類題を生成」を押すと、ここに似た問題が並びます。
キーはこの端末の localStorage にのみ保存。例題は生成時に Google(Gemini) へ送信されます・保存されません。AIが解いた答えなので稀に誤りがあります。

設定

取得: Google AI Studio → Create API key

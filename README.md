# QuizCraft

例題から類題を作って演習。間違いだけ、忘れた頃にもう一度。

解きたい問題（例題）のテキストや写真を貼ると、AI（Gemini）が **同じ分野・同じ形式の新しい演習問題（類題）** を生成し、自分で解いた **解答・解説つき** で出題します。解いて間違えた問題だけが、忘れた頃（翌日→3日後→1週間後…）に再出題されます。

暗記科目向けに、ノート・教科書から用語や定義の **確認テスト** を作るモードも選べます。

## 特徴

- **類題モード（既定）** — 例題に似た新問をAIが作成。数字を変えただけの焼き直しではなく、設定・文脈を変えた別問題を、最終解答と途中式つきで出題（数学・理科向け）
- **LaTeX 数式表示** — 問題文・選択肢・解答・解説を MathJax で数式描画。化学式・反応式（`\ce{}`）にも対応
- **確認テストモード** — ノートの範囲から4択・穴埋め・記述を生成。全問に教材原文の引用（根拠）つき（文系・暗記向け）
- **おかしい問題はワンタップ破棄** — AIの解答が怪しければその場で確認して破棄でき、以後出題されません
- **間隔反復（Leitner式）** — 間違えた問題ほど頻繁に、解けた問題ほど間隔を空けて再出題
- **完全ローカル** — 問題と学習状況はブラウザ（localStorage）にだけ保存。サーバーなし・アカウントなし
- **例題はテキストでも写真でも** — 教科書・問題集の写真からも作問（Gemini vision）

## 使い方

1. `index.html` をブラウザで開く（またはGitHub Pages版にアクセス）
2. 設定から [Google AI Studio](https://aistudio.google.com/app/apikey) の無料APIキーを登録
3. 「＋ 問題を作る」で「類題を作る」を選び、例題を貼る／写真を追加 → 生成 → 解く → 翌日また開く

## 開発

- 単一HTMLファイル（ビルドなし・依存なし）
- テスト: `node tests/srs.test.mjs`（間隔反復ロジックの単体テスト）

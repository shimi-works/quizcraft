# QuizCraft

ノートから小テスト。間違いだけ、忘れた頃にもう一度。

講義ノートや教科書のテキスト・写真を貼ると、AI（Gemini）が確認テストを生成。解いて間違えた問題だけが、忘れた頃（翌日→3日後→1週間後…）に再出題されます。

## 特徴

- **根拠つき出題** — 全問題に教材原文からの引用（根拠）が付き、答え合わせのたびに表示。AIの作問が信用できるかその場で検証でき、おかしい問題はワンタップで破棄できます
- **間隔反復（Leitner式）** — 間違えた問題ほど頻繁に、覚えた問題ほど間隔を空けて出題
- **完全ローカル** — 問題と学習状況はブラウザ（localStorage）にだけ保存。サーバーなし・アカウントなし
- **教材はテキストでも写真でも** — 手書きノートの写真からも作問（Gemini vision）

## 使い方

1. `index.html` をブラウザで開く（またはGitHub Pages版にアクセス）
2. 設定から [Google AI Studio](https://aistudio.google.com/app/apikey) の無料APIキーを登録
3. 「＋ 教材から作る」に講義ノートを貼って生成 → 解く → 翌日また開く

## 開発

- 単一HTMLファイル（ビルドなし・依存なし）
- テスト: `node tests/srs.test.mjs`（間隔反復ロジックの単体テスト）

# QuizCraft

例題を渡すと、似た演習問題（類題）を解答つきで一発生成。

解きたい問題（例題）のテキストや写真を貼って押すだけで、AI（Gemini）が **同じ分野・同じ形式の新しい演習問題** を数問作り、AI自身が解いた **最終解答と途中式** を付けて並べます。数式は LaTeX で表示。

「AIに類題を作って」とチャットに毎回打つより速く、写真からそのまま作れて、結果がきれいに数式表示されるのが取り柄です。

## 特徴

- **例題 → 類題を一発生成** — 数字を変えただけの焼き直しではなく、設定・文脈を変えた別問題を、最終解答＋途中式つきで
- **LaTeX / 化学式表示** — 問題文・解答・解説を MathJax で数式描画（化学反応式 `\ce{}` 対応）
- **解いてから答え合わせ** — 答えは各問たたんで表示。解いてから開ける。全開/全閉も一括
- **写真からもOK** — 教科書・問題集の写真から作問（Gemini vision・複数枚可）
- **もっと作る** — 同じ例題から既出とかぶらない追加の類題を継ぎ足し
- **コピー** — 1問ずつ／問題だけ（解答なし）／全部まとめてコピー（LaTeXそのまま）
- **印刷（演習プリント）** — 問題ページ＋別ページの解答・解説で紙に出して机で解ける
- **スマホのホーム画面に追加できる**（PWA）— 写真から作る使い方が1タップで
- **完全ローカル** — APIキー・履歴はブラウザ（localStorage）にだけ保存。サーバーなし・アカウントなし

AIが数式を解いて出題する以上、難問では稀に答えを外すことがあります。おかしいと感じたら解説で検算できます。

## 使い方

1. `index.html` をブラウザで開く（またはGitHub Pages版にアクセス）
2. 設定から [Google AI Studio](https://aistudio.google.com/app/apikey) の無料APIキーを登録
3. 例題を貼る／写真を追加 → 「類題を生成」（Ctrl+Enterでも可）
4. 速さ優先なら設定で `gemini-2.5-flash-lite`、難しい計算の精度優先なら `gemini-flash-latest`

## 開発

- 単一HTMLファイル（ビルドなし・依存なし）＋ PWA用の manifest / sw.js / icons。数式描画のみ MathJax を CDN から読み込み
